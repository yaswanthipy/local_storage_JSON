# Enhanced Todo List with Search & localStorage

## Features
- Add new tasks
- Mark tasks as completed (✔ button)
- Remove tasks from the list (✖ button)
- Search tasks in real time
- Tasks are saved in localStorage
- Each task contains:
  - unique ID
  - text
  - completion status

## How to Use
1. Enter a task in the input box and click **Add Task**
2. Click **✔** to mark completed
3. Click **✖** to delete a task
4. Use the search bar to filter tasks instantly
5. Reload the page — tasks will still be there (saved in localStorage)

## Technologies Used
- HTML
- CSS
- JavaScript
- localStorage API
